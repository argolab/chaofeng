安装¶

代码托管在 github 上面:

$ git clone https://github.com/argolab/chaofeng.git
$ sudo python setup.py install

完成。

如果不希望装进库里面，也可以这样:

$ git clone https://github.com/argolab/chaofeng.git
$ cp -r chaofeng/chaofeng path-to-you-project-use-chaofeng

恩，或许你需要一个简单的测试:

$ cd path-to-git-clone
$ python test.py
